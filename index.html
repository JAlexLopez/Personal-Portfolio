ABOUT ME

Hi! My name is Alex Lopez and I'm a Frontend Web Developer who is passionate about developing the web through clean, semantic, and scalable code. I'm intrigued by coding and excited to continue learning new technologies.

I initially discovered programming in high school after watching The Social Network. The movie sparked a curiosity and I signed up for the game development class in my sophomore year. There, I learned to make 2D games such as Flappy Bird and Super Mario using Visual Basic, which I picked up very quickly and consider a very fun programming language. After that I took one semester of C++ which was a fair bit more challenging.

After working a few odd jobs after high school and attending college part time, I discovered Web Development and fell in love immediately. I decided to start teaching myself HTML, CSS, and JavaScript while taking other courses. The next semester I enrolled in the Web Development Program at my college and there I added to my knowledge of the aforementioned languages and even took another C++ course.
